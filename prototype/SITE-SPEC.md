## 1. サイト全体構成

### 1.1 ページ構成

```
リンゴラボ（仮）
├── トップページ（ブランド拠点・記事への導線）
├── 記事一覧（カテゴリー別）
│   ├── 温活
│   ├── 美容
│   └── 食事
├── 個別記事ページ
├── 商品ページ（りんごティー）
├── アプリページ（プロトタイプ）
└── お問い合わせ（将来実装）
```

### 1.2 主要導線

```
【記事導線（最優先）】
トップページ → 記事一覧（#articles） → 個別記事 → アフィリエイトリンク

【ブランド導線】
トップページ → プロフィール（#profile） → 信頼感の醸成

【商品・アプリ導線（副次的）】
トップページ → 商品ティーザー → 商品ページ
トップページ → アプリティーザー → アプリページ
ナビゲーション → 商品/アプリページへ直接アクセス
```

**設計思想**: トップページの主目的は「記事を読んでもらうこと」。商品・アプリは控えめに紹介。

### 2.4 視覚リズム・アイコン規約（v2.1）

| ルール | 内容 |
|--------|------|
| **背景** | メインコンテンツは `landing-band--a`（#FAFAF8）と `landing-band--b`（#FFF）を**交互**に使用。ヒーローはグラデのみ例外。 |
| **区切り** | 各帯の上辺に `landing-band` の **1px 実線**（茶系8%）。破線は使わない。 |
| **セクションの意味** | 見出し直前に `.section-kicker`（英字ラベル + Font Awesome）で意味を付与。 |
| **アイコン** | **OS絵文字・装飾絵文字は禁止**。Font Awesome **Solid** を基本とし、色は `muted-red` または `brown` で統一。 |
| **商品だけベージュ帯** | 禁止。温かさは **内側カード**（`.teaser-product-inner`）のみで表現。 |
| **CTAの終端** | 締めCTAは **パネル + 信頼バー**の二段構造。 |

### 9.3 コンセプトコピー候補（検討中）

現行案「整う生活は、置くだけで始まる」は**今後見直し検討**

**方向性**:
- Apple的：シンプル・洗練・無駄のない
- ブランド価値：頑張らない温活、専門性、信頼性
- ターゲット：忙しい女性、丁寧な暮らしへの憧れ

**候補案**（例）:
- 「1日1個で、整う習慣」
- 「りんごから始まる、温活習慣」

## 11. SEO・アクセシビリティ要件

### 11.1 SEO基本要件

- **タイトルタグ**: 32文字以内、キーワード含む
- **メタディスクリプション**: 120文字以内、魅力的な説明
- **見出し構造**: h1（1つ）→ h2 → h3の階層
- **画像alt属性**: 全画像に説明文
- **内部リンク**: 関連記事への自然なリンク
- **パンくずナビ**: 全ページに実装

### 11.2 アクセシビリティ要件

- **キーボード操作**: Tab/Enterで全操作可能
- **フォーカス表示**: `:focus-visible`で明示
- **スクリーンリーダー**: `aria-label`適切に設定
- **色コントラスト**: WCAG AA準拠（4.5:1以上）
- **レスポンシブ**: モバイルファーストデザイン
