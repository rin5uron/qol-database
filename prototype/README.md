# 🍎 リンゴラボ（仮） - トップページ試作版

## 📁 このディレクトリについて

リンゴラボ（仮）のトップページHTML試作版です。

### 目的
- デザインの確認
- デザイナーへのイメージ共有
- WordPress構築前のプロトタイプ検証

---

## 🎨 含まれるセクション

1. **ナビゲーション**（固定ヘッダー）
2. **ヒーローエリア**（メインビジュアル・ヘッドライン・CTA）
3. **信頼性・社会的証明**（管理栄養士バッジ・実績）
4. **ブランドコンセプト**（Problem-Solution）
5. **3つの特徴**（整える・巡る・学ぶ）
6. **管理栄養士プロフィール**
7. **記事一覧**（カテゴリータブ・記事カード）
8. **最終CTA**
9. **フッター**

---

## 🚀 使い方

### ローカルで確認
```bash
# qol-databaseディレクトリで
cd prototype
open index.html  # Macの場合
```

ブラウザで `index.html` を開くだけで確認できます。

## 🎨 カラーパレット

| カラー | 用途 | カラーコード |
|--------|------|-------------|
| **くすみ赤** | CTAボタン・アクセント | `#C97D7D` |
| **クリーム色** | カード背景・セクション背景 | `#F5EFE6` |
| **焦茶** | 見出し・強調テキスト | `#6B4423` |
