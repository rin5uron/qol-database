## ロゴデザイン方針

### ブランドロゴの役割

**リンゴラボのアイデンティティを視覚化する**

- **認識性**: 一目で「リンゴラボ」と分かる
- **記憶性**: 覚えやすく、印象に残る
- **拡張性**: 様々なメディアで使用可能

---

### デザインコンセプト

**「りんご + シンプル + 温かみ」**

- **りんご**: ブランドの象徴を明確に
- **シンプル**: 北欧暮らし風のミニマルデザイン
- **温かみ**: くすみピンクで優しい印象

---

### デザイン方向性

#### 案A: りんごアイコン + テキストロゴ（推奨）

**イメージ**:
```
 🍎  リンゴラボ
```

**特徴**:
- シンプルで認識しやすい
- りんごのシンボルが明確
- 北欧暮らし風のミニマル

**メリット**:
- ✅ ブランド象徴（りんご）が明確
- ✅ 拡張性が高い（アイコンだけでも使える）
- ✅ モダンで洗練されている

**デメリット**:
- ❌ Apple社との混同リスク

---

#### 案B: テキストロゴのみ

**イメージ**:
```
リンゴラボ
```

**特徴**:
- 極限までシンプル
- フォントで個性を出す
- 游ゴシック or Noto Sans JP

**メリット**:
- ✅ 視認性が高い
- ✅ Apple社との混同回避
- ✅ 実装が簡単

**デメリット**:
- ❌ りんごの象徴が弱い
- ❌ 記憶に残りにくい

---

#### 案C: りんごをモチーフにした抽象ロゴ

**イメージ**:
```
 ◯   リンゴラボ
  ˙  （葉っぱ）
```

**特徴**:
- りんごを抽象化
- オリジナリティが高い
- デザイナーの介入が必要

**メリット**:
- ✅ オリジナリティ
- ✅ Apple社との差別化

**デメリット**:
- ❌ 制作コストが高い
- ❌ りんごと認識されにくい可能性

---

### 推奨案

**案A: りんごアイコン + テキストロゴ**

**理由**:
1. ブランド象徴（りんご）が明確
2. シンプルで拡張性が高い
3. 自作可能（Canva、Figmaで作成）

## ロゴ仕様

### ロゴタイプ（テキスト）

**フォント**:
- 游ゴシック Medium
- または Noto Sans JP Medium (500)

**文字サイズ**:
- 大: 48px
- 中: 32px
- 小: 24px

**カラー**:
- プライマリ: `--color-brown` (#6b5344)
- セカンダリ: `--color-muted-pink` (#d8a7a0)

**文字間隔**:
- `letter-spacing: 0.05em`

---

### アイコン（りんご）

**デザイン**:
- シンプルなりんごのシルエット
- 葉っぱ付き
- フラットデザイン

**カラー**:
- りんご本体: `--color-muted-pink` (#d8a7a0)
- 葉っぱ: `--color-brown` (#6b5344)

**サイズ**:
- 大: 64×64px
- 中: 48×48px
- 小: 32×32px

---

### ロゴ構成

**横並び（推奨）**:
```
[りんごアイコン] リンゴラボ
```

**縦並び**:
```
[りんごアイコン]
リンゴラボ
```

**アイコンのみ**:
```
[りんごアイコン]
```

## 使用ガイドライン

### ロゴの使い方

**✅ OK**:
- ブランドカラーで使用
- クリアスペース確保
- 適切なサイズで使用

**❌ NG**:
- 色の変更（規定外）
- 変形（引き伸ばし、圧縮）
- 回転
- グラデーション追加
- 影の追加

---

### キャラクターの使い方

**✅ OK**:
- プロフィールカード
- 記事内の補足
- SNS投稿

**❌ NG**:
- 商用利用（グッズ販売）
- 改変（表情変更、服装変更）
- 他サイトでの無断使用

---

### ファイル形式

**ロゴ**:
- SVG（推奨）: 拡大縮小に強い
- PNG（透過）: 汎用性が高い
- JPG: 背景付き

**キャラクター**:
- PNG（透過）: 推奨
- SVG: イラストレーター形式

## 制作方法

### ロゴ制作

**ツール**:
- Canva（無料）
- Figma（無料）
- Adobe Illustrator（有料）

**手順**:
1. りんごアイコンを作成（円 + 楕円 + 葉）
2. テキストを配置（游ゴシック Medium）
3. カラーを適用（くすみピンク + ブラウン）
4. SVG/PNGで書き出し

---

### キャラクター制作

**ツール**:
- Canva（無料、イラスト素材組み合わせ）
- Adobe Illustrator（有料、オリジナル制作）
- ココナラ（外注、3,000円〜）

**手順**:
1. ラフスケッチ
2. 線画作成
3. 着色
4. PNG（透過）で書き出し
