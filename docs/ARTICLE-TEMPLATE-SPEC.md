## 記事タイプ定義

### タイプA: 体験レビュー記事（主力）

**目的**: 実体験をもとに、読者の不安を解消し、自然な導線でアフィリエイトへ誘導

**例**:
- 「りんちゃんが3ヶ月試した、LAVA体験のリアル」
- 「よもぎ蒸しサロンに通って分かった、冷え性への効果」

**構成**:
1. 導入（共感ポイント）
2. りんちゃんの体験談
3. 管理栄養士の視点（エビデンス）
4. 使い方・続け方のコツ
5. 自然な導線（アフィリエイト）

**文字数**: 3,000-5,000字

---

### タイプB: ハウツー記事

**目的**: 検索意図に応え、実用的なアドバイスを提供

**例**:
- 「よもぎ蒸しの始め方と続けるコツ」
- 「冷え性改善！温活習慣5選」

**構成**:
1. 導入（問題提起）
2. 解決策の提示（3-5ステップ）
3. 管理栄養士の補足（エビデンス）
4. よくある質問
5. まとめ + 導線

**文字数**: 2,500-4,000字

---

### タイプC: 商品解説記事

**目的**: 商品の詳細を解説し、購入を後押し

**例**:
- 「無添つくりおき｜忙しいママに最適な理由」
- 「管理栄養士が選ぶ、温活サプリ3選」

**構成**:
1. 導入（商品選びの悩み）
2. 商品の特徴・成分
3. 管理栄養士の評価（エビデンス）
4. 使用シーン・体験談
5. 購入リンク

**文字数**: 2,000-3,500字

## 記事構成テンプレート

### 【体験レビュー記事】テンプレート

#### 1. 導入（共感ポイント）200-300字

```markdown
## こんなお悩み、ありませんか？

- 冷え性が辛い
- 忙しくて自分の時間がない
- 続けられない

私もLAVAに通い始める前は、同じ悩みを抱えていました。
```

**ポイント**:
- 読者の悩みを3-5個列挙
- 「私も」でりんちゃんが共感
- 問いかけで引き込む

#### 2. りんちゃんの体験談（800-1,200字）

```markdown
## りんちゃんがLAVAに通い始めたきっかけ

管理栄養士として、体温と代謝の関係は理解していました。でも、仕事が忙しくて運動する時間がない…。

そんなとき、友人に誘われて体験レッスンへ。
```

**ポイント**:

## 見出し構成ルール

### H1（タイトル）

**形式**:
- 北欧暮らし式: 「りんちゃんが3ヶ月試した、〜のリアル」
- ラフリー式: 「〜の始め方と続けるコツ」

**ルール**:
- 50-60文字以内
- 主要キーワードを含む
- 数値で具体性（3ヶ月、5選）
- 感情ワード（リアル、最適、おすすめ）

### H2（大見出し）

**推奨見出し**:
1. こんなお悩み、ありませんか？
2. りんちゃんが〜を始めたきっかけ
3. 管理栄養士が解説：〜の効果
4. 無理なく続けるコツ
5. 私が選んだのは〜

### H3（中見出し）

**ルール**:
- H2の補足
- 具体的な項目（「週2回、固定曜日に予約」）
- 読者の疑問に答える形式

## E-E-A-T対策

### Experience（経験）

**実装方法**:
- りんちゃんの実体験を必ず盛り込む
- 具体的な数値（3ヶ月、週2回、体温0.5℃UP）
- 写真: 生活シーン

### Expertise（専門性）

**実装方法**:
- 「管理栄養士が解説」セクションを必須化
- 栄養学・生理学の知見
- 専門用語を分かりやすく説明

### Authoritativeness（権威性）

**実装方法**:
- プロフィールに管理栄養士資格を明記
- 資格番号・所属団体を記載（フッター）
- エビデンス（論文・ガイドライン）を引用

### Trustworthiness（信頼性）

**実装方法**:
- 正確な情報
- エビデンスで裏付ける
- 免責事項（「個人の感想です」）
- 薬機法・景表法を遵守

## 収益導線設計

### アフィリエイトリンクの配置

**位置**:
1. 記事中盤（H2-3の後）: テキストリンク
2. 記事末尾: ボタンCTA
3. サイドバー: バナー（PC表示のみ）

**形式**:
- **テキストリンク**: 「LAVAの詳細はこちら」
- **ボタンCTA**: 「LAVAの体験レッスンを予約する」
- **バナー**: 画像 + テキスト

### CTAコピー

**良い例**:
- 「体験レッスンを予約する」
- 「〜の詳細を見る」
- 「〜を試してみる」

**悪い例**:
- 「クリックはこちら」
- 「詳細」
- 「公式サイト」

### 免責事項

```markdown
※個人の感想です。効果には個人差があります。
※アフィリエイト広告を利用しています。
```

## 薬機法・景表法コンプライアンス

### 絶対にNGな表現

| NG表現 | 理由 | 推奨表現 |
|--------|------|---------|
| 「治る」「効く」 | 医薬品的な効能効果 | 「期待できる」「サポートする」 |
| 「100%」「必ず」 | 誇大広告 | 「〜が報告されています」 |
| 「No.1」「最安値」 | 根拠のない最上級表現 | 「人気の〜」 |
| 「改善する」 | 断定表現 | 「〜に役立つ可能性がある」 |

### 推奨される表現

| シーン | 推奨表現 |
|--------|---------|
| 効果を伝える | 「〜に役立つと言われています」 |
| 体験談 | 「個人の感想です。効果には個人差があります」 |
| 論文引用 | 「〜という研究報告があります」 |
| 栄養素の働き | 「〜の働きをサポートする」 |

### チェックリスト

- [ ] 「治る」「効く」「改善する」を使っていないか
- [ ] 「100%」「必ず」「絶対」を使っていないか
- [ ] 根拠のない最上級表現（No.1、最高）を使っていないか
- [ ] 免責事項（「個人の感想です」）を記載したか
- [ ] アフィリエイト広告である旨を明記したか
